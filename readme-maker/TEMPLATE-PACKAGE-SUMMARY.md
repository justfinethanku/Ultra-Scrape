## How to Use

### Quick Start (5 minutes)
1. Copy `README-TEMPLATE.md` to your project as `README.md`
2. Do a find/replace: `[PROJECT_NAME]` ‚Üí Your actual project name
3. Fill in the `{{BRACES}}` placeholders in the hero section

5. Remove all HTML comments (`<!-- ... -->`)
